# Traffic-Control-FPGA
Este repositorio contiene el código para un Sistema de Gestión de Tráfico en FPGA, diseñado para controlar el flujo de tráfico en una rotonda con 4 semáforos para vehículos y 4 semáforos para peatones. El proyecto utiliza una máquina de estados implementada en Verilog para gestionar el cambio de semáforos y garantizar la seguridad en la rotonda.
